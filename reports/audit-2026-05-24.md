## Why these surfaced under "5 Minutes Max"

The frontend filter sends `prefs.time = "5min"`. A card is in the 5-min bucket if any of:

1. `timeCommitment` string contains "5" or "quick" (matches "5 min", "< 5 min", "5 minutes", "quick task")
2. `quickAction: true` AND `timeCommitment` is empty or the legacy "< 1 hour"
3. No `timeCommitment` set AND category default is `5min` (BOYCOTT / MENTAL HEALTH / FUNDING / BOOST)
